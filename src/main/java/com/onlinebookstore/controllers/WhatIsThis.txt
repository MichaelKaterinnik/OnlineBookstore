Класи, які відповідають за обробку HTTP-запитів та управління взаємодією між користувачем та програмою.

Обробка запитів від користувачів та взаємодія зі змінними стану системи.

Класи, що відповідають за обробку запитів користувача (наприклад, UserController, BookController,
OrderController), реалізовані згідно з шаблоном проектування MVC (Model-View-Controller).

BookController: контролер для відображення списку книг та детальної інформації про книгу;
UserController: контролер для реєстрації та авторизації користувачів;
OrderController: контролер для здійснення замовлень та перегляду історії замовлень користувача.
